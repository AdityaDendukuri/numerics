2D Poisson Solver

Header: #include "pde/poisson.hpp"
Namespace: num::pde

Solves \(-\nabla^2 u = f\) on \((0,1)^2\) with homogeneous Dirichlet boundary conditions, discretised on an \(N \times N\) interior grid with mesh spacing \(h = 1/(N+1)\). Two variants are provided: a finite-difference solver with \(O(h^2)\) error and a spectral solver with machine-precision error for \(f\) in the DST eigenbasis. Both run in \(O(N^2 \log N)\) via the Discrete Sine Transform.

Reference: J. Demmel, Parallel Spectral Methods: Fast Fourier Transform with Applications, CS267 Lecture 20, UC Berkeley, 2025.

Problem formulation

Find \(u : (0,1)^2 \to \mathbb{R}\) such that

\[ -\nabla^2 u = f \quad \text{on } (0,1)^2, \qquad u = 0 \text{ on } \partial\Omega. \]

Discretise on the \(N \times N\) interior grid \((x_i, y_j) = (ih, jh)\), \(i,j = 1,\ldots,N\), using the standard 5-point Laplacian. The discrete system is

\[ L_2\, \mathbf{u} = h^2\, \mathbf{f}, \]

where \(L_2 = L_1 \otimes I + I \otimes L_1\) and \(L_1\) is the \(N \times N\) tridiagonal matrix \(\mathrm{tridiag}(-1,\,2,\,-1)\).

Eigenvalue decomposition

\(L_1\) is symmetric and its eigenvectors are the sine modes

\[ F_{jk} = \sin\!\left(\frac{jk\pi}{N+1}\right), \quad j,k = 1,\ldots,N, \]

so \(L_1 = F\,D\,F^T\) with diagonal eigenvalues

\[ D_k = 2\left(1 - \cos\frac{k\pi}{N+1}\right), \quad k = 1,\ldots,N. \]

Because \(L_2 = L_1 \otimes I + I \otimes L_1\), its eigenvalues are \(D_j + D_k\) for all pairs \((j,k)\). Applying the 2-D DST to both sides of the discrete system decouples it into \(N^2\) independent scalar equations:

\[ (D_j + D_k)\,\hat{u}_{jk} = h^2\,\hat{f}_{jk}. \]

Algorithm (FD solver)

  1. Compute \(\hat{F} = \mathrm{DST2D}(h^2 f)\).
  2. Divide pointwise: \(\hat{U}_{jk} = \hat{F}_{jk} / (D_j + D_k)\).
  3. Recover \(u = \mathrm{IDST2D}(\hat{U})\).

The inverse DST-I satisfies \(\mathrm{IDST} = \tfrac{2}{N+1}\,\mathrm{DST}\), so step 3 is another forward DST followed by scaling by \(\bigl(\tfrac{2}{N+1}\bigr)^2\).

Error: \(O(h^2)\), from the FD truncation in the eigenvalue \(D_k = 2(1-\cos(k\pi/(N+1)))\). The ratio \(\|u_h - u\|_\infty / \|u_{h/2} - u\|_\infty \to 4\) as \(h \to 0\).

Algorithm (spectral solver)

Replace the FD eigenvalue \(D_k\) with the exact eigenvalue \((k\pi)^2\) of the continuous operator \(-d^2/dx^2\).

Normalization. The unnormalised DST satisfies \(F \cdot F = \tfrac{N+1}{2}\,I\), so the continuous spectral coefficient of \(f\) is

\[ \tilde{f}_{jk} = \frac{4}{(N+1)^2}\,\hat{F}_{j-1,\,k-1}. \]

Dividing by \((j^2+k^2)\pi^2\) and expanding the reconstruction sum shows that \(u = \mathrm{DST2D}(\hat{U})\) without an extra \(\bigl(\tfrac{2}{N+1}\bigr)^2\) factor, where

\[ \hat{U}_{jk} = \frac{4}{(N+1)^2\,\pi^2\,(j^2+k^2)}\,\hat{F}_{jk}. \]

Algorithm:

  1. Compute \(\hat{F} = \mathrm{DST2D}(f)\) (no \(h^2\) prefactor).
  2. Multiply: \(\hat{U}_{jk} = \hat{F}_{jk} \cdot \dfrac{4}{(N+1)^2\,\pi^2\,(j^2+k^2)}\).
  3. Recover \(u = \mathrm{DST2D}(\hat{U})\).

Error: zero up to floating-point rounding for any \(f\) expressible as a finite sum of DST eigenfunctions at the grid resolution. For smooth \(f\) with many active modes, convergence is exponential in \(N\).

DST-I via complex FFT

The unnormalised Discrete Sine Transform of type I of an \(N\)-point real vector \(x\) is

\[ X[k] = \sum_{j=1}^{N} x[j]\,\sin\!\left(\frac{jk\pi}{N+1}\right), \quad k = 1,\ldots,N. \]

It is computed via a complex FFT on the odd-extended sequence of length \(M = 2(N+1)\):

\[ y = \bigl[0,\; x[1],\; \ldots,\; x[N],\; 0,\; -x[N],\; \ldots,\; -x[1]\bigr]. \]

The FFT of \(y\) satisfies \(Y[k] = -2i\,X[k]\), so

\[ X[k] = -\frac{1}{2}\,\mathrm{Im}\bigl(Y[k]\bigr), \quad k = 1,\ldots,N. \]

Size constraint: \(M = 2(N+1)\) must be a power of two for the radix-2 backend, which requires \(N = 2^p - 1\) (e.g. 7, 15, 31, 63, 127, 255).

The 2-D DST is \(F^T \cdot A \cdot F\), applied by sweeping DST-I over columns then rows. The total cost is \(O(N^2 \log N)\).

The f matrix uses row-major indexing: f(i, j) is the RHS value at \((x,y) = ((i+1)h,\,(j+1)h)\).

Worked example

Test problem with two active modes:

\[ f(x,y) = 2\pi^2\sin(\pi x)\sin(\pi y) + 5\pi^2\sin(2\pi x)\sin(\pi y), \]

\[ u(x,y) = \sin(\pi x)\sin(\pi y) + \sin(2\pi x)\sin(\pi y). \]

#include "numerics.hpp"
#include <cmath>
using namespace num;
int main() {
const int N = 31; // 2^5 - 1
const double h = 1.0 / (N + 1);
const double pi = M_PI;
Matrix f(N, N), u_exact(N, N);
for (int i = 0; i < N; ++i) {
const double x = (i + 1) * h;
for (int j = 0; j < N; ++j) {
const double y = (j + 1) * h;
f(i, j) = (2*pi*pi*std::sin(pi*x) + 5*pi*pi*std::sin(2*pi*x)) * std::sin(pi*y);
u_exact(i, j) = (std::sin(pi*x) + std::sin(2*pi*x)) * std::sin(pi*y);
}
}
Matrix u_fd = pde::poisson2d_fd(f, N); // O(h^2) error
Matrix u_spec = pde::poisson2d(f, N); // machine precision
}
