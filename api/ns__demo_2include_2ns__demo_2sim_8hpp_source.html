1/// @file include/ns_demo/sim.hpp
2/// @brief 2-D incompressible Navier-Stokes, periodic MAC grid
3///
4/// Algorithm: Chorin projection method
5/// 1. Semi-Lagrangian advection -> u*, v*
6/// 2. Build RHS: rhs = -div(u*)/dt
7/// 3. CG pressure solve: (-Lap)p = rhs (positive-definite, Backend::omp
8/// inner products)
9/// 4. Project: u = u* - dt*gradp
10///
11/// Grid (NxN cells, domain [0,1]^2):
12/// u[i,j] -- x-velocity at face (i*h, (j+1/2)*h) i,j in [0,N)
13/// v[i,j] -- y-velocity at face ((i+1/2)*h, j*h) i,j in [0,N)
14/// p[i,j] -- pressure at centre ((i+1/2)*h, (j+1/2)*h)
15///
16/// Storage: row-major, index = i*N + j.
17/// All boundaries are periodic.
