Detailed Description

Givens plane rotation: G = [c, s; -s, c].

Constructed so that G * [a; b]^T = [r; 0] with r = hypot(a, b).

auto g = GivensRotation::from(h_jj, h_jj1); // zero H[j,j+1] in GMRES
g.apply(g_j, g_j1); // rotate RHS vector too
