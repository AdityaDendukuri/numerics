Detailed Description

Boltzmann acceptance probability helpers for Metropolis MCMC.

boltzmann_accept(dE, beta) – min(1, exp(-beta*dE)) inline, branch-free exp make_boltzmann_table(dEs, beta) – precompute table for a discrete dE set

Typical usage (Ising model):

// Replace: boltz[si][ni] = (dE <= 0) ? 1.0 : exp(-beta*dE)
boltz[si][ni] = num::markov::boltzmann_accept(dE, beta);
