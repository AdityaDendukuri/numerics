Symmetric Jacobi Eigendecomposition (eig_sym)

Problem Statement

Given \(A \in \mathbb{R}^{n \times n}\) symmetric ( \(A = A^T\)), find the spectral decomposition:

\[A = V \Lambda V^T, \qquad V^TV = I, \quad \Lambda = \operatorname{diag}(\lambda_1, \ldots, \lambda_n)\]

The Jacobi method is an iterative similarity transform that drives all off-diagonal entries to zero by repeatedly applying Givens (plane) rotations.

Mathematical Foundation

Givens Rotation in the \f$(p,q)\f$ Plane

The Givens rotation \(G(p, q, \theta)\) is the identity with a \(2 \times 2\) rotation block at rows/columns \(p\) and \(q\):

\[G_{pp} = c,\quad G_{pq} = -s,\quad G_{qp} = s,\quad G_{qq} = c, \qquad c = \cos\theta,\ s = \sin\theta\]

The similarity transform \(A' = G^T A G\) modifies only rows/columns \(p\) and \(q\):

\[A'_{pp} = c^2 A_{pp} - 2cs\, A_{pq} + s^2 A_{qq}\]

\[A'_{qq} = s^2 A_{pp} + 2cs\, A_{pq} + c^2 A_{qq}\]

\[A'_{pq} = A'_{qp} = (c^2 - s^2) A_{pq} + cs(A_{pp} - A_{qq}) \quad \leftarrow \text{want } = 0\]

\[A'_{rp} = A'_{pr} = c\, A_{rp} - s\, A_{rq}, \quad r \neq p, q\]

\[A'_{rq} = A'_{qr} = s\, A_{rp} + c\, A_{rq}, \quad r \neq p, q\]

Choosing \f$\theta\f$ to Zero \f$A'_{pq}\f$

Set \(A'_{pq} = 0\):

\[(c^2 - s^2) A_{pq} + cs(A_{pp} - A_{qq}) = 0\]

Let \(t = \tan\theta\) and \(\tau = (A_{qq} - A_{pp})/(2A_{pq})\):

\[t^2 + 2\tau t - 1 = 0 \quad \Longrightarrow \quad t = \frac{\operatorname{sign}(\tau)}{|\tau| + \sqrt{1 + \tau^2}}\]

Taking the smaller root \(|t| \leq 1\) (i.e., \(|\theta| \leq \pi/4\)) minimizes the perturbation to other off-diagonal entries. Then:

\[c = \frac{1}{\sqrt{1 + t^2}}, \qquad s = ct\]

The formula avoids catastrophic cancellation when \(\tau \gg 1\): in that case \(t \approx 1/(2\tau) \approx 0\), so \(c \approx 1\) and \(s \approx 0\) – almost no rotation is needed.

Algorithm: Cyclic Jacobi

Visit every off-diagonal pair \((p, q)\) in a fixed cyclic order (row-by-row sweep). The off-diagonal Frobenius norm \(\sigma_{\text{off}} = \sqrt{2\sum_{p<q} A_{pq}^2}\) is the convergence monitor:

function eig_sym(A, tol, max_sweeps):
n <- A.rows
V <- I_n
for sweep = 1 to max_sweeps:
off <- sqrt(2 * sum_{p<q} A[p,q]^2)
if off < tol: break
for p = 0 to n-2:
for q = p+1 to n-1:
if |A[p,q]| < 1e-15: continue
tau <- (A[q,q] - A[p,p]) / (2 * A[p,q])
t <- sign(tau) / (|tau| + sqrt(1 + tau^2))
c <- 1 / sqrt(1 + t^2); s <- c * t
// Update diagonal and zero (p,q)
A'[p,p] <- c^2*A[p,p] - 2cs*A[p,q] + s^2*A[q,q]
A'[q,q] <- s^2*A[p,p] + 2cs*A[p,q] + c^2*A[q,q]
A[p,q] <- A[q,p] <- 0
// Update off-diagonal rows and columns r != p,q
for r != p,q:
(A[r,p], A[r,q]) <- (c*A[r,p] - s*A[r,q], s*A[r,p] + c*A[r,q])
A[p,r] <- A[r,p]; A[q,r] <- A[r,q]
// Accumulate eigenvectors: V <- V * G
for r = 0..n-1:
(V[r,p], V[r,q]) <- (c*V[r,p] - s*V[r,q], s*V[r,p] + c*V[r,q])
lambda <- diagonal of A
return lambda, V (sort ascending)

Cost per sweep: \(\frac{n(n-1)}{2}\) rotations \(\times\) \((4n + 4n)\) flops/rotation \(= O(4n^3)\) per sweep.

Convergence

The off-diagonal Frobenius norm satisfies:

\[\sigma^2_{\text{off}}(A_{k+1}) \leq \left(1 - \frac{2}{n(n-1)}\right) \sigma^2_{\text{off}}(A_k) \qquad \text{(linear phase)}\]

When \(\sigma_{\text{off}}\) becomes small, convergence transitions to the quadratic ultimate phase:

\[\sigma_{\text{off}}(A_{k+1}) \leq C\, \sigma^2_{\text{off}}(A_k)\]

In practice: 5-10 sweeps for double precision with well-separated eigenvalues; <= 15 sweeps for clustered eigenvalues.

Monotonicity of Diagonal Entries

Each rotation decreases \(\sigma_{\text{off}}\) by exactly \(2A_{pq}^2\):

\[\sigma^2_{\text{off}}(A') = \sigma^2_{\text{off}}(A) - 2A_{pq}^2\]

The Frobenius norm \(\|A\|_F^2 = \sigma^2_{\text{diag}} + \sigma^2_{\text{off}}\) is preserved (similarity transform), so the diagonal Frobenius norm \(\sigma^2_{\text{diag}} = \sum_i A_{ii}^2\) monotonically increases.

Variants

Threshold Jacobi

Skip rotation if \(|A_{pq}| < \varepsilon_k \cdot \sigma_{\text{off}}\) (adaptive threshold). Reduces work in early sweeps when many entries are already small.

One-Sided Jacobi

Apply rotations from the right only: \(A_{k+1} = A_k G\). Column norms \(\|A[:,j]\|\) converge to singular values. This is the basis of the one-sided Jacobi SVD – see svd-onesided-jacobi.md.

Parallel Jacobi (Tournament Ordering)

The cyclic sweep visits pairs sequentially. Instead, use a tournament schedule: pair up all \(n/2\) disjoint pairs simultaneously (no pair shares an index -> independent rotations). After \(n-1\) rounds, all pairs have been visited:

\[n = 8: \quad \text{Round 1}: (0,1),(2,3),(4,5),(6,7); \quad \text{Round 2}: (0,2),(1,4),(3,6),(5,7); \quad \ldots\]

Each round requires a synchronization barrier, but rotations within a round are embarrassingly parallel -> GPU-friendly. This is the Jacobi-Winograd parallel scheme.

LAPACK dsyev uses:

  1. Tridiagonalization via Householder: \(\frac{4}{3}n^3\) flops (DSYTRD)
  2. QR iteration on the tridiagonal: \(O(n^2)\) flops (DSTEQR)
  3. Back-transform eigenvectors: \(2n^3\) flops (DORMTR)

Our Jacobi implementation is simpler and directly verifiable, but LAPACK's tridiagonalize-then-QR is 2-3 \(\times\) faster for large \(n\).

Performance Optimization

Current Implementation

The rotation update has two phases:

  • Diagonal update: 4 scalar flops
  • Off-diagonal update: \(4n\) flops (two length- \(n\) rotation loops)
  • Eigenvector accumulation: \(4n\) flops

Total per rotation: \(8n + 4 \approx 8n\) flops. Per sweep: \(\frac{n(n-1)}{2} \times 8n = 4n^3(1 - 1/n)\) flops.

SIMD for the Rotation Update (Column-Major A)

With column-major storage, columns \(p\) and \(q\) are contiguous. The off-diagonal update is a 2D rotation of two column vectors:

__m256d vc = _mm256_broadcast_sd(&c);
__m256d vs_neg = _mm256_broadcast_sd(&s_neg); // -s
__m256d vs = _mm256_broadcast_sd(&s);
for (idx r = 0; r < n; r += 4) {
__m256d ap = _mm256_load_pd(&A(r, p));
__m256d aq = _mm256_load_pd(&A(r, q));
// new_p = c*ap - s*aq, new_q = s*ap + c*aq
__m256d new_p = _mm256_fmadd_pd(vc, ap, _mm256_mul_pd(vs_neg, aq));
__m256d new_q = _mm256_fmadd_pd(vs, ap, _mm256_mul_pd(vc, aq));
_mm256_store_pd(&A(r, p), new_p);
_mm256_store_pd(&A(r, q), new_q);
}

Peak throughput: 2 FMA + 2 store = 10 instructions per 4 elements. For \(n = 1024\): one rotation takes 256 FMA cycles ~= 85 ns at 3 GHz.

For row-major storage, exploit symmetry: update row \(p\) and row \(q\) of \(A\) (same elements by symmetry). Keep a separate column-major buffer for \(V\).

Exploiting Symmetry in the Update

Store only the upper triangle of \(A\) (since \(A\) is symmetric). The rotation update for row \(r\) (off-diagonal update) writes:

\[A_{rp}' = c\, A_{rp} - s\, A_{rq}, \qquad A_{rq}' = s\, A_{rp} + c\, A_{rq}\]

For \(r < p\): access \(A[r,p]\) and \(A[r,q]\) (upper triangle) – both accessible. For \(r > q\): access \(A[p,r]\) and \(A[q,r]\) (stored as \(A[r,p]^T\)) – still accessible.

This halves memory traffic: update \(n/2\) entries per rotation on average instead of \(n\).

Blocked Jacobi (Brent-Luk)

For large \(n\), process rotations in blocks of size \(b\). Instead of scalar Givens rotations in the \((p, q)\) plane, apply a \(b \times b\) orthogonal transformation in the \((p:p+b, q:q+b)\) subspace:

  1. Extract the \(2b \times 2b\) subblock \(B = \begin{pmatrix} A_{PP} & A_{PQ} \\ A_{QP} & A_{QQ} \end{pmatrix}\)
  2. Diagonalize \(B\) via small \(2b \times 2b\) Jacobi -> orthogonal matrix \(G_{2b}\)
  3. Apply \(G_{2b}\) to the full matrix: \(A \leftarrow G^T A G\) as two DGEMM calls

The dominant cost becomes DGEMM (BLAS-3): arithmetic intensity \(O(b)\) instead of \(O(1)\) for scalar FMA. With \(b = 64\) and \(n = 1024\): achieve near-peak GFLOP/s on the trailing DGEMM portion.

Cache Behaviour

The rotation simultaneously accesses columns \(p\) and \(q\). With column-major storage, both are contiguous: \(2n \times 8\) bytes. For \(n = 2048\): each column is 16 KB, two columns = 32 KB -> fits in L1 (32 KB typical). For \(n = 4096\): 64 KB -> L2 traffic.

Column-pair buffer: copy columns \(p, q\) to 32-byte-aligned scratch buffers, update scratch, write back. Eliminates potential L1 conflict misses when \(p, q\) map to the same cache set.
