1/// @file include/nbody/sim.hpp
2/// @brief Gravitational N-body simulation using num::ode_verlet /
3/// num::ode_rk4_2nd.
4///
5/// NBodySim wraps the ODE module to provide a per-frame step() interface.
6/// Each call to step(dt) runs exactly one time step of the chosen integrator:
7///
8/// Verlet (symplectic, default) -- ode_verlet: O(h^2), bounded energy
9/// error RK4 (non-symplectic) -- ode_rk4_2nd: O(h^4), secular
10/// energy drift
11///
12/// Both integrators share the same AccelFn / SymplecticResult interface, so
13/// switching between them requires no restructuring of the state.
14///
15/// Positions and velocities are stored as flat num::Vector with stride 2
16/// ([x0,y0, x1,y1, ...]). Use num::Vec2View for readable index-free access.
